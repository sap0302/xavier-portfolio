# Xavier Simpson Portfolio Site

Static portfolio website focused on medical imaging data systems, DICOM/PACS workflows, Horos/OsiriX viewer enablement, database architecture, SQL performance, AWS infrastructure, ETL/reporting, and secure healthcare-data operations.

## Files

- `index.html` — portfolio page
- `styles.css` — responsive styling
- `script.js` — mobile navigation and reveal animations
- `resume.pdf` — downloadable resume

## Deployment

### GitHub Pages
1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Go to **Settings → Pages**.
4. Select the main branch and root folder.
5. Save and open the generated GitHub Pages URL.

### Netlify
1. Go to Netlify.
2. Drag and drop this folder into the Netlify deploy area.
3. Netlify will publish the site automatically.

### Vercel
1. Create a new Vercel project.
2. Import the repository or upload the static files.
3. Use the default static site settings.

## Note

The site frames existing database architecture and infrastructure experience toward medical imaging workflows. Customize wording further if you want to claim direct production experience with specific tools, facilities, or clinical systems.
